## ✨ O QUE VOCÊ VAI VER

### **1. Esforço vs Desempenho** ⚡
```
Bolhas coloridas mostrando:
• Seu esforço percebido (X)
• Seu desempenho (Y)
• Qual treino é mais eficiente
```

### **2. Produtividade** ⏱️
```
Barras horizontais mostrando:
• Qual treino é mais rentável em tempo
• Resultado por minuto investido
```

### **3. Desempenho por Esporte** 🏃
```
Barras verticais mostrando:
• Seu esporte mais forte
• Seu esporte mais fraco
• Onde focar para melhorar
```

### **4. Posição no Ranking** 📈
```
Barra de progresso mostrando:
• Seu percentil (0-100%)
• Comparação com todos os usuários
• Status: Excelente, Acima, Média, Abaixo
```

### **5. Relação Peso** ⚖️
```
Cards informativos mostrando:
• Seu peso atual
• Sua melhora de desempenho
• Status geral
```
